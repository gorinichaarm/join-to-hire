Test task JointoHire by Gor Knyazyan!

First you need the plugin to be active on your WordPress site.
Steps to activate (manual installation)
1. Download and unzip the plugin to your computer.
2. Upload the join-to-hire folder to the /wp-content/plugins/ directory of your WordPress site
3. Activate the plugin through the ‘Plugins’ menu in WordPress
4. And now you can see the ‘Contact Forms’ new added menu on the left side of your WP dashboard.

You need to create new form, by clicking on the Add new submenu item below the ‘Contact Forms’.

One step left and you are done. 
Copy the shortcode of your created form, and paste it in any page/post to see the result.
Enjoy it, hope you'll be satisfied.

You also need smtp to be able to send emails from local environment.
